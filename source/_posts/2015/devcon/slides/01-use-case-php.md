## What "*install PHP5*" means

```bash
# Ubuntu
$ apt-get install php5 libapache2-mod-php5 php5-curl
$ php --version
> PHP 5.5.9 (cli)

# CentOS 6
$ yum install php curl curl-devel
$ php --version
> PHP 5.4.16 (cli)
```
